Aideation is a productivity and knowledge management platform built as a modern alternative to **Notion**. It provides an intuitive environment for content creation, task management, and collaboration, offering seamless integration with **Clerk** for authentication, **generative AI** for content generation, and real-time collaborative tools.

<br/>

<h1 className="text-2xl font-semibold" > Tech Stack </h1>

- **Clerk** 
- **Generative AI** (`@google/generative-ai`, `ai`) 
- **Neon Database** (`@neondatabase/serverless`)
- **Radix UI** (`@radix-ui/react-dialog`, `@radix-ui/react-separator`)
- **TanStack Query** (`@tanstack/react-query`)

### Key Features

- **Collaborative Workspace**: Work together on documents and tasks, with real-time updates.

<br/>


- **Content Management**: Organize notes, ideas, and documents in a structured way, just like in Notion.
<br/>
- **Generative AI**: Use AI to auto-generate content for notes, tasks, and ideas.
<br/>
- **Authentication**: Secure and easy user management with **Clerk**.
<br/>
- **Rich Text Editing**: Create and edit rich-text documents with the **Tiptap editor**.
<br/>
- **Task Management**: Organize your tasks and to-do lists efficiently.
<br/>
- **Real-time Collaboration**: Sync changes instantly across all users.
<br/>
- **Database**: Store and manage user content with **Neon Database**.
